编程的范式

为什么要懂一点编程 ​

分类 ​

根据代码如何执行, 可以分为两大类 1. 解释型语言，解释器解释执行 2. 编译型语言，编译成机器语言后执行

其他分类

  • 系统编程语言: C, C++
  • 应用编程语言: Java, C#
  • 脚本语言，编程语言的子集，属于解释型语言: python, javascript, shell
  • web开发语言: html, css, php
  • 科学计算数据分析: R, MATLAB, python
  • ...

shell ​

  • shell 是计算机操作系统的用户界面, 是用户与内核交互的接口
  • shell 是一个命令解释器,
  • 分为：①命令行界面（CLI）；② 图形用户界面（GUI）
  • 例如: macos的finder, linux的窗口管理器, windows的explorer和dos shell都属于GUI式的shell
  • 无特殊说明一般指命令行界面shell

windows的shell ​

  • CMD: command prompt
  • powershell
  • WSL

linux的shell ​

  • sh: 最早的unix shell
  • bash: 最常用的linux shell
  • fish, zsh 等是一些现代的shell, 用户友好, 补全功能更智能.
  • Shell、Bash、CMD、PowerShell 的区别_powershell bash-CSDN博客

shell编程 ​

  • shell 提供了一种脚本语言, 组合多个命令, 实现复杂逻辑和自动化操作, 甚至编写一些简单的应用.
  • 不同shell的解释器不同, 脚本文件语法不同, 例如:
    • windows cmd的批处理文件.bat
    • windows powershell的脚本文件.ps1
    • linux 的脚本文件.sh .bsah
  • shell从脚本中逐行读取命令，并在系统中搜索这些命令，然后编译器将程序转换为机器可读的形式，即可执行文件.
  • /etc/shells 文件给出了当前Linux系统上支持的shell类型
  • linux用户的默认shell设置记录在 /etc/passwd 文件中
  • 最常用的bash
