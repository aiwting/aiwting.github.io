静态网站的概述 ​

静态网站的定义: 网页内容固定，每个页面都有预先编写或者及时生成的 HTML 文件，用户在访问时，服务器直接将这些文件发送给浏览器进行显示。

静态网站的特点是内容预先设定, 交互性差, 数据不变, 不需要数据库的支持, 技术实现简单, 加载速度快, 安全性高. 适用于内容更新不频繁的个人博客、企业宣传页、产品展示页等.

实现方式: 静态网站的技术实现简单, 即使采用传统的方法, 也仅需要 HTML、CSS 和 JavaScript 三大前端语言即可写出漂亮的网站页面. 如今出现了众多的 #开源 静态网站生成器, 能够方便的将预先定义好的模板、数据和内容转化为静态 HTML 网页。

静态网站生成器（Static Site Generator, SSG）, 能够把你的文档、内容（通常为 Markdown 文件）生成可发布成网站的 html 文档, 特别适合搭建一个个人博客或是给项目做说明文档网站.

静态站点生成器 (SSG) ​

GitHub 上流行、优秀的开源静态网站生成器举例如下.

Vitepress/Vuepress ​

  • Vitepress 基于 Vite 和 Vue, 能够将markdown文档编译为html文档.
  • Vuepress是其的上一代产品, 已经过时, Vue项目团队已经将主要精力放在Vitepress上.
  • 特点: Vitepress 主要用于快速搭建美观、易用的技术文档网站, 附带的默认主题已经足够简洁美观(见官方文档); 支持增强的 Markdown 语法(见官方文档); 支持搜索功能, 明暗主题, 层级分类.
  • GitHub: vitepress
  • 官方文档: VitePress-zh

Hexo ​

  • Heox 使用 Node.js 开发,
  • 特点: Hexo 是一个快速、简洁且高效的博客框架, 主题和插件丰富; 使用 Markdown（或其他标记语言）解析文章，在几秒内即可利用靓丽的主题生成静态网页。
  • 官网: Hexo: 快速、简洁且高效的博客框架
  • GitHub: hexo
  • 教程: Easy-Hexo
  • 首先需要安装 Node.js 和 Git, 安装和使用示例:
sh
#在当前目录下局部安装
npm install hexo 
#运行hexo
npx hexo <command>
#初始化
npx hexo init <foldername>
cd <foldername>
npm install
#本地预览部署
npx hexo server
#生成静态网页
npx hexo generate

Docsify ​

  • Docsify 基于 Vue，偏向于文档网站，动态实时地将markdown文档转换为html网页
  • 特点: Docsify 只有一个 index.html，所有的配置都在这个文件中, 只有在打开网页的时候才开始生成其它html页面.
  • GitHub: docsify
  • 官方文档: A magical documentation site generator

Hugo ​

  • Hugo 使用 Go 语言开发, 号称"世界上最快的网页生成器"
  • GitHub: hugo
  • 官网: The world’s fastest framework for building websites

其他 ​

  • Jekyll
  • 官网: Transform your plain text into static websites and blogs
  • Gatsby
  • MkDocs: 基于 Python 编写
    • MkDocs（Markdown Documents）是一个快速、简单的静态网站生成器，用于将多个 Markdown 文档组织起来构建成有层次、美观的文档站点。

部署方式 ​

这里介绍一些将你的静态网站部署上线的方法.

GitHub Pages ​

GitHub Pages 是 GitHub 提供的一项免费静态网站托管服务, 允许用户直接从 GitHub 仓库托管和发布个人、组织或项目的静态网站。

特点: 无需额外付费，只要有 GitHub 账号就能使用该服务; 可以结合git版本控制工具使用, 方便的更新和推送网站代码; 只需把静态文件（如 HTML、CSS、JavaScript）推送到指定的 GitHub 仓库，GitHub Pages 就会自动构建并发布网站; 提供免费的github.io域名, 也支持使用自定义域名.

Read and Docs ​

  • Read the Docs 是一个开源的文档托管和管理平台, 有社区版和商用版
  • Read the Docs 默认使用Sphinx作为其文档生成工具，并提供了专门的Sphinx主题
  • Sphinx是一个基于Python的文档生成工具, 支持多种输出格式，包括HTML、PDF、LaTeX, reStructuredText和Markdown格式.
  • 官网: Read the Docs: Documentation simplified
