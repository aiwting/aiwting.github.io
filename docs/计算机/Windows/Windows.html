命令行界面 ​

Windows 有两个命令行工具

命令提示符 cmd.exe ​

  • CMD 是Windows系统的一种命令行工具, 属于Windows系统的基本组件, 是一种直接操作系统的途径
  • 功能: 文件和目录管理, 系统配置和维护, 网络管理, 故障排除, 自动化任务
  • 原则上 cmd 命令不区分大小写

Windows PowerShell ​

  • 是一款功能强大的命令行界面（CLI）和脚本语言
  • 兼容所有传统CMD命令, 扩展了cmd的功能(cmdle命令, 格式: "动词-名词")
  • 允许将几个命令组合起来放到文件里执行, 实现文件级的重用, 也就是说有脚本的性质, 比 .bat 脚本更加强大.
  • 相关特定文件类型有脚本文件 (.ps1)、脚本数据文件 (.psd1) 和脚本模块文件 (.psm1)。
  • 在PowerShell命令行输入脚本文件全名（xxx.ps1）即可运行
  • 脚本文件.ps1和.bat的区别：
    • .ps1 是 PowerShell 脚本文件的扩展名，
    • .bat 是 Windows 批处理脚本文件的扩展名，
  • Windows对PowerShell脚本默认的执行策略是：不允许执行，即Restricted。
  • 使用get-executionpolicy 可以在PowerShell命令行中查看执行策略，如果结果是Restricted，那表示禁止执行脚本。需要通过更改执行策略： Set-ExecutionPolicy RemoteSigned 或者 set-executionpolicy -executionpolicy unrestricted
  • 如果执行ps1文件，Powershell解释器会逐行解释并执行它的的语句
  • PowerShell 在线教程 – PowerShell 中文博客 (pstips.net)

常用Windows命令 ​

findstr ​

  • 搜索文件中的文本，类似于linux的grep
  • findstr | Microsoft Learn

chcp ​

  • Windows操作系统默认编码格式是GBK（936）
  • 查看cmd当前编码方式chcp, 会输出: 活动代码页：936
  • chcp 65001会修改cmd编码方式为UTF-8

注册表 regedit ​

  • 注册表编辑器（Regedit.exe），可以用来查看和维护注册表
  • Windows注册表(Registry)实质上是一个庞大的数据库，它存储着下面这些内容：用户计算机软、硬件的有关配置和状态信息，应用程序和资源管理器外壳的初始条件、首选项和卸载数据；计算机的整个系统的设置和各种许可，文件扩展名与应用程序的关联，硬件的描述、状态和属性；计算机性能记录和底层的系统状态信息，以及各类其他数据。
  • 当前系统的注册表数据库在C:\Windows\System32\config
  • .ini文件：程序的配置文件。从Windows95系统开始，注册表开始全面替代ini文件，成为Windows系统中用于管理配置系统运行参数的核心数据库。这种集中管理的缺点是：不利于删除，软件的所有文件不存在于一个文件夹下，散落在系统的各个角落

WSL ​

WSL 与 Windows 之间有很好的互操作性，可以进行文件共享和相互访问。

  1. 文件访问：你可以在 Linux 发行版的终端中访问 Windows 文件系统，路径为 /mnt/<盘符>。例如，/mnt/c/ 是访问 C 盘的路径。
  2. Windows 访问 Linux 文件：打开 “资源管理器”，在地址栏输入 \\wsl$，即可访问 Linux 文件系统。
  3. 与 Windows 应用程序集成：WSL 支持在 Linux 中运行一些基于文本的 Windows 应用程序，例如编辑器和浏览器。

Windows启动管理器 ​

Windows Boot Manager（Windows启动管理器）是引导操作系统启动的一个工具，当我们的电脑安装了多个操作系统时，我们就可以调出启动管理器，在电脑启动时就会在显示操作系统的选择界面，让我们选择自己想要的系统来启动。 ![[Pasted image 20240709153423.png|500]]

安全模式 ​

安全模式的工作原理是在不加载第三方设备驱动程序的情况下启动电脑，使电脑运行在系统最小模式，这样用户就可以方便地检测与修复计算机系统的错误。 包括： 启用安全模式。 通过网络启用安全模式。 使用命令提示符启用安全模式。 ![[Pasted image 20240709153546.png|400]]

PE ​

Windows PE (WinPE，Windows preinstallation environment)，是一个小型操作系统，用于安装、部署和修复 Windows 桌面版、Windows Server 和其他 Windows 操作系统。衍生了很多的个性化工具，包括：磁盘管理、备份还原、UFEI引导修复、硬件检测、账户密码破解、数据恢复…… 可从 U 盘、CD/DVD/ISO 或虚拟硬盘驱动器启动 WinPE。 WinPE可以被第三方修改，有各种版本的PE：微PE、Edgeless、优启通、FirPE、Hiren's BootCD PE ![[Pasted image 20240709110148.png|300]] PE运行在内存中（会将部分内存转化为临时磁盘） Windows PE (WinPE) | Microsoft Learn

Visual Studio ​

Visual Studio是微软公司推出的一款集成开发环境（IDE），广泛应用于计算机程序、网站、Web应用、Web服务和移动应用的软件开发。主要用于适用于Windows平台

Windows Kits是一个专为Windows应用程序开发设计的工具包。Windows Kits文件夹通常位于Visual Studio的安装目录下，包含了编译器、调试器、性能分析器、资源编辑器等工具，以及Windows SDK和驱动程序开发工具，确保开发者能够创建高效、可靠且符合标准的Windows应用程序。
