图像的存储方式 ​

  • 除了绘画以外，图片的产生方式：摄像机和计算机

模拟图像 ​

  • ‌模拟图像, 记录或传输信息的信号是连续的, 又称连续图像. 通过某种物理量（‌如光、‌电等）‌的强弱变化来记录图像亮度信息。例如:
    • 模拟电视的图像就是通过光的强弱传输的.
    • 胶片图像上记录的像点是无限稠密的.
    • 人眼所见的图像也是连续的强弱光信号产生在视网膜上的.
  • 随着对光的认识和光学定律的提出，通过一定的光学透镜系统形成的图像是模拟图像，最初通过感光材料（胶片）记录图像。
  • 后来随着电子技术的发展，使用传感器（传感器是由大量的光电二极管组成的阵列）记录光学系统产生的图像，再经过模数转换(ADC)得到数字图像。

数字图像 ​

  • 使用计算机存储的数据来记录图像上各点的亮度信息。‌这种记录方式依赖于计算机技术，‌使得图像的处理和编辑变得更加方便和灵活。‌
  • 光电效应：光→光电二极管（传感器）→电信号→ADC→数字信号
  • 光电传感器: CCD（电荷耦合器件）和CMOS（互补金属氧化物半导体）是数码相机和其他图像捕捉设备中常用的两种图像传感器技术。这两种类型的传感器都包含数百万个光电二极管组成的阵列，每个光电二极管称为像素（Pixel），相机像素值即代表传感器上光电二极管的数量。光电二极管只能感知光线强弱（0-255），无法区分颜色，需要获得彩色图像需要使用色彩滤波器。
  • 色彩成像原理：相机传感器表面通常覆盖有一层滤色器阵列，其中最常见的是拜耳阵列（Bayer filter）。拜耳阵列（RGGB）由红、绿、蓝三种颜色的滤色器交替排列组成，每个滤色器只允许对应颜色的光通过。这种排列方式使得每个像素点只能捕捉到一种颜色的光信息，但相邻的像素点会捕捉到不同颜色的光信息。分别存储三基色（红绿蓝）即可生成一个图片文件。
  • 拜耳阵列（RGGB）：每个像素点（光电二极管）被一个色块覆盖，每个色块被四个其它两种色块包围，经过插值法可以计算出该点的RGB数值。
  • 通道：RGB彩色图片在计算机中存储时，会使用R、G、B三个通道（三基色）来分别表示红色、绿色和蓝色的强度，这三个通道共同决定了每个像素的颜色。RGBA彩色图像使用红色（Red）、绿色（Green）、蓝色（Blue）和透明度（Alpha）四个通道
  • 位深：又称位分辨率（bit per pixel），代表精度，与音频位深类似。常见8位， 16位， 24位， 32位等。图像深度决定了图像的每个像素可能的颜色数，或可能的灰度级数。例如，RGBA彩色图像每个像素用R,G,B三个分量加一个透明度表示，每个分量用8位，透明度用8位，像素深度为32位。RGB彩色图片每个像素用R,G,B三个分量，每个分量用8位，像素深度为24位。
  • 透明度：透明度是指图像中像素的透明程度，它决定了图像中的像素与背景或其他图像混合的方式。在RGBA颜色模型中，Alpha通道用于表示透明度，其值范围通常从0（完全透明）到255（完全不透明）。透明度允许图像以不同的方式与背景或其他图像结合，创造出丰富的视觉效果。
  • 灰度：灰度是图像亮度的表示，它是介于黑色和白色之间的一系列灰色调。在灰度图像（即黑白图片）中，每个像素的亮度值决定了其灰度等级，而颜色的色相和饱和度信息则被去除。灰度图像通常用于图像处理中的预处理步骤，如边缘检测、图像分割等。灰度图像可以通过RGB图像转换而来，这种转换基于RGB三个颜色通道的不同权重或算法来实现。
  • 真彩色：当使用24位存储图像时，能够记录 224=16777216 种颜色，超过了人眼可分辨的颜色数量，称为真彩图片.

显示设备 ​

  • 常见屏幕：CRT显示器、液晶显示器(LCD)、等离子体显示器、有机发光二极管显示器(LED)，涉及电子或光子的激发与控制，以及颜色的合成与刷新。其中LCD显示器最常见
  • LCD屏又细分为VA、IPS、TN屏，区别在于液晶层分子排列和偏转不同
  • 像素点：最小的可控制光点，是画面中最小单位的基本构成元素。显示器的分辨率由像素点决定，具体表现为屏幕上横向和竖向的像素点数量。例如，一块分辨率为1920x1080的显示器，表示它在水平和垂直方向上分别有1920和1080个像素点，总共可以显示2,073,600个像素。

常见图像文件格式 ​

  • JPEG（.jpg、.jpeg） ：这是一种广泛使用的有损压缩格式，适用于存储照片和其他真彩色图像。JPEG文件体积小，适合网络传输和打印应用。
  • PNG（.png） ：这是一种无损压缩的位图格式，支持透明背景，常用于网页设计和数字图形应用。PNG格式比GIF更高效，文件尺寸较小。
  • GIF（.gif） ：这是一种基于LZW算法的无损压缩格式，支持动画和透明度，适用于简单的动画、图标和表情符号。
  • BMP（.bmp） ：这是Windows系统下的标准位图格式，未经过压缩，文件体积较大，但图像质量高，适合对图像质量要求较高的场景。
  • TIFF（.tif、.tiff） ：这是一种灵活的位图格式，支持多种编码方法，常用于专业印刷和出版领域，提供高质量的图像存储。
  • WebP（.webp） ：由Google开发的图像格式，结合了高质量的压缩和小文件大小，适用于Web上的图像展示。
  • RAW（.raw） ：这是一种未处理的原始图像格式，直接从相机传感器获取数据，保留了最多的图像信息，适合后期处理。
  • PSD（.psd） ：这是Adobe Photoshop的专用格式，支持图层、通道和路径信息，常用于图像编辑和设计。
  • SVG（.svg） ：这是一种基于XML的矢量图形格式，可以任意缩放而不失真，适用于Web图形页面的设计。
  • EPS（.eps） ：这是一种跨平台的标准格式，用于矢量和光栅图像的存储，常用于印刷行业。
