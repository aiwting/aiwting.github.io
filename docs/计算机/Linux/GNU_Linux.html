д»Җд№ҲжҳҜGNU/Linux вҖӢ

  • GMUжҳҜдёҖеҘ—ж“ҚдҪңзі»з»ҹпјҢжҳҜдёҖз»„дҫӣз”ЁжҲ·дҪҝз”Ёзҡ„иҪҜд»¶зЁӢеәҸпјҡ
    • GNUеҗҚеӯ—зҡ„еҗ«д№үжҳҜдёҖдёӘйҖ’еҪ’зј©еҶҷпјҢд»ЈиЎЁвҖңGNU's Not UnixвҖқпјҲGNUдёҚжҳҜUnixпјүгҖӮиҝҷдёӘеҗҚз§°з”ұиҮӘз”ұиҪҜд»¶еҹәйҮ‘дјҡпјҲFree Software Foundation, FSFпјүзҡ„еҲӣе§ӢдәәзҗҶжҹҘеҫ·В·ж–ҜжүҳжӣјпјҲRichard StallmanпјүжҸҗеҮәпјҢж—ЁеңЁеҲӣе»әдёҖдёӘе®Ңе…ЁиҮӘз”ұзҡ„Unixе…је®№ж“ҚдҪңзі»з»ҹпјҢзӣ®ж ҮжҳҜејҖеҸ‘дёҖдёӘиҮӘз”ұзҡ„ж“ҚдҪңзі»з»ҹпјҢејәи°ғз”ЁжҲ·еңЁдҪҝз”ЁиҪҜд»¶ж—¶еә”дә«жңүиҮӘз”ұпјҢеҢ…жӢ¬иҝҗиЎҢгҖҒеӨҚеҲ¶гҖҒеҲҶеҸ‘гҖҒеӯҰд№ гҖҒдҝ®ж”№е’Ңж”№иҝӣиҪҜд»¶зҡ„иҮӘз”ұгҖӮ
    • GNUж“ҚдҪңзі»з»ҹз”ұи®ёеӨҡиҮӘз”ұиҪҜд»¶з»„жҲҗпјҢиҝҷдәӣиҪҜд»¶йҒөеҫӘGNUйҖҡз”Ёе…¬е…ұи®ёеҸҜиҜҒпјҲGPLпјүпјҢиҝҷжҳҜдёҖз§Қе№ҝжіӣдҪҝз”Ёзҡ„иҮӘз”ұиҪҜд»¶и®ёеҸҜиҜҒгҖӮеӣ жӯӨGNUдёҚд»…жҳҜдёҖдёӘж“ҚдҪңзі»з»ҹпјҢе®ғиҝҳеҢ…еҗ«еӨ§йҮҸзҡ„иҮӘз”ұиҪҜд»¶еҢ…пјҢеҰӮзј–иҜ‘еҷЁGCCгҖҒж–Үжң¬зј–иҫ‘еҷЁEmacsзӯүпјҢиҝҷдәӣиҪҜд»¶йғҪжҳҜз”ұGNUйЎ№зӣ®жҲ–з¬¬дёүж–№еҸ‘еёғзҡ„гҖӮйҖҡиҝҮиҝҷдәӣиҪҜд»¶пјҢз”ЁжҲ·еҸҜд»ҘиҮӘз”ұең°дҪҝз”Ёи®Ўз®—жңәиҖҢдёҚеҝ…жӢ…еҝғдҫөзҠҜзүҲжқғжҲ–еҸ—еҲ°е•ҶдёҡиҪҜд»¶зҡ„йҷҗеҲ¶гҖӮ
  • Linuxд№ҹжҳҜж“ҚдҪңзі»з»ҹпјҢдҪҶжҳҜжңҖеә•еұӮзҡ„иҪҜд»¶пјҢзӣҙжҺҘдёҺзЎ¬д»¶дәӨдә’пјҢз§°дёәж“ҚдҪңзі»з»ҹзҡ„еҶ…ж ёйғЁеҲҶпјҢиҙҹиҙЈз®ЎзҗҶзЎ¬д»¶иө„жәҗе’Ңдёәеә”з”ЁзЁӢеәҸжҸҗдҫӣеә•еұӮжңҚеҠЎгҖӮ
  • LinuxдёҺGNUйЎ№зӣ®жңүзқҖеҜҶеҲҮзҡ„е…ізі»пјҡе°Ҫз®ЎLinuxеҶ…ж ёжң¬иә«е№¶дёҚжһ„жҲҗдёҖдёӘе®Ңж•ҙзҡ„ж“ҚдҪңзі»з»ҹпјҢдҪҶе…¶дёҺGNUйЎ№зӣ®зҡ„з»“еҗҲдҪҝеҫ—е®ғиғҪеӨҹжҸҗдҫӣе®Ңж•ҙзҡ„зі»з»ҹеҠҹиғҪпјҢLinuxеҶ…ж ёиҙҹиҙЈзЎ¬д»¶з®ЎзҗҶпјҢиҖҢGNUе·Ҙе…·й“ҫе’ҢиҪҜд»¶еҲҷжҸҗдҫӣдәҶз”ЁжҲ·з•ҢйқўгҖҒзј–зЁӢе·Ҙе…·е’Ңе…¶д»–еә”з”ЁзЁӢеәҸгҖӮ
  • GNUж“ҚдҪңзі»з»ҹйҖҡеёёдёҺLinuxеҶ…ж ёз»“еҗҲдҪҝз”ЁпјҢиҜһз”ҹдәҶдё–з•ҢдёҠжңҖиҮӘз”ұзҡ„ GNU/Linuxж“ҚдҪңзі»з»ҹгҖӮ
  • LinuxеҸ‘иЎҢзүҲпјҲеҰӮUbuntuгҖҒDebianгҖҒFedoraзӯүпјүйҖҡеёёиў«з§°дёәвҖңLinuxвҖқпјҢдҪҶе®һйҷ…дёҠе®ғд»¬жӣҙеҮҶзЎ®ең°еә”иҜҘиў«з§°дёәвҖңGNU/LinuxвҖқпјҢеӣ дёәе®ғд»¬дҫқиө–дәҺGNUйЎ№зӣ®жҸҗдҫӣзҡ„еӨ§йҮҸиҪҜд»¶гҖӮ

Linuxзҡ„з»“жһ„ вҖӢ

LinuxеҸҜеҲ’еҲҶдёәд»ҘдёӢеӣӣйғЁеҲҶпјҡ

  1. LinuxеҶ…ж ёпјҡеҶ…ж ёжҺ§еҲ¶зқҖи®Ўз®—жңәзі»з»ҹдёҠзҡ„жүҖжңүзЎ¬д»¶е’ҢиҪҜд»¶

    • зі»з»ҹеҶ…еӯҳз®ЎзҗҶ
    • иҪҜд»¶зЁӢеәҸз®ЎзҗҶ
    • зЎ¬д»¶и®ҫеӨҮз®ЎзҗҶ
    • ж–Үд»¶зі»з»ҹз®ЎзҗҶ

  2. GNUе·Ҙе…·пјҡдёҖдәӣзі»з»ҹе·Ҙе…·пјҢз”ЁдәҺжү§иЎҢдёҖдәӣж ҮеҮҶеҠҹиғҪпјҢжҜ”еҰӮжҺ§еҲ¶ж–Үд»¶е’ҢзЁӢеәҸгҖӮйҖҡеёёе°ҶLinuxеҶ…ж ёе’ҢGNUе·Ҙе…·зҡ„з»“еҗҲдҪ“з§°дёәLinux

    • GNU coreutilsиҪҜд»¶еҢ…пјҲж ёеҝғGNUе·Ҙе…·пјүз”ұдёүйғЁеҲҶжһ„жҲҗпјҡ
      • з”Ёд»ҘеӨ„зҗҶж–Үд»¶зҡ„е·Ҙе…·
      • з”Ёд»Ҙж“ҚдҪңж–Үжң¬зҡ„е·Ҙе…·
      • з”Ёд»Ҙз®ЎзҗҶиҝӣзЁӢзҡ„е·Ҙе…·
    • shellпјҡеЈізЁӢеәҸshellжҳҜз”ЁжҲ·ж“ҚдҪңзі»з»ҹзҡ„дёҖдёӘз•ҢйқўжҲ–е№іеҸ°пјҢжҳҜдёҖз§Қзү№ж®Ҡзҡ„дәӨдә’ејҸе·Ҙе…·пјҢе°ұжҳҜе‘Ҫд»ӨиЎҢжЁЎејҸгҖӮе®ғдёәз”ЁжҲ·жҸҗдҫӣдәҶеҗҜеҠЁзЁӢеәҸгҖҒз®ЎзҗҶж–Үд»¶зі»з»ҹдёӯзҡ„ж–Үд»¶д»ҘеҸҠиҝҗиЎҢеңЁLinuxзі»з»ҹдёҠзҡ„иҝӣзЁӢзҡ„йҖ”еҫ„гҖӮshellзҡ„ж ёеҝғжҳҜе‘Ҫд»ӨиЎҢжҸҗзӨәз¬ҰгҖӮе‘Ҫд»ӨиЎҢжҸҗзӨәз¬ҰжҳҜshellиҙҹиҙЈдәӨдә’зҡ„йғЁеҲҶгҖӮе®ғе…Ғи®ёдҪ иҫ“е…Ҙж–Үжң¬е‘Ҫд»ӨпјҢз„¶еҗҺи§ЈйҮҠе‘Ҫд»ӨпјҢе№¶еңЁеҶ…ж ёдёӯжү§иЎҢгҖӮ
      • shellеҢ…еҗ«дәҶдёҖз»„еҶ…йғЁе‘Ҫд»ӨпјҢз”Ёиҝҷдәӣе‘Ҫд»ӨеҸҜд»Ҙе®ҢжҲҗиҜёеҰӮеӨҚеҲ¶ж–Үд»¶гҖҒз§»еҠЁж–Үд»¶гҖҒйҮҚе‘ҪеҗҚж–Үд»¶гҖҒжҳҫзӨәе’Ңз»Ҳжӯўзі»з»ҹдёӯжӯЈиҝҗиЎҢзҡ„зЁӢеәҸзӯүж“ҚдҪңгҖӮ
      • shellд№ҹе…Ғи®ёдҪ еңЁе‘Ҫд»ӨиЎҢжҸҗзӨәз¬Ұдёӯиҫ“е…ҘзЁӢеәҸзҡ„еҗҚз§°пјҢе®ғдјҡе°ҶзЁӢеәҸеҗҚдј йҖ’з»ҷеҶ…ж ёд»ҘеҗҜеҠЁе®ғгҖӮ
      • д№ҹеҸҜд»Ҙе°ҶеӨҡдёӘshellе‘Ҫд»Өж”ҫе…Ҙж–Үд»¶дёӯдҪңдёәзЁӢеәҸжү§иЎҢгҖӮиҝҷдәӣж–Үд»¶иў«з§°дҪңshellи„ҡжң¬гҖӮдҪ еңЁе‘Ҫд»ӨиЎҢдёҠжү§иЎҢзҡ„д»»дҪ•е‘Ҫд»ӨйғҪеҸҜж”ҫиҝӣдёҖдёӘshellи„ҡжң¬дёӯдҪңдёәдёҖз»„е‘Ҫд»Өжү§иЎҢгҖӮиҝҷдёәеҲӣе»әйӮЈз§ҚйңҖиҰҒжҠҠеҮ дёӘе‘Ҫд»Өж”ҫеңЁдёҖиө·жқҘе·ҘдҪңзҡ„е·Ҙе…·жҸҗдҫӣдәҶдҫҝеҲ©гҖӮ
      • Linuxзі»з»ҹдёҠпјҢйҖҡеёёжңүеҘҪеҮ з§ҚLinux shellеҸҜз”ЁгҖӮеңЁ/etc/shellзӣ®еҪ•дёӢжүҖжңүLinuxеҸ‘иЎҢзүҲй»ҳи®Өзҡ„shellйғҪжҳҜbash shellгҖӮ
        • bash shellз”ұGNUйЎ№зӣ®ејҖеҸ‘пјҢиў«еҪ“дҪңж ҮеҮҶUnix shellвҖ”вҖ”Bourne shellпјҲд»ҘеҲӣе»әиҖ…зҡ„еҗҚеӯ—е‘ҪеҗҚпјүзҡ„жӣҝд»Је“ҒгҖӮ
        • е…¶д»–shellзҡ„з®ҖеҚ•д»Ӣз»Қ | е…¶е®ғshell | жҸҸиҝ° | | --------- | -------------------------------------------------------- | | ash | дёҖз§ҚиҝҗиЎҢеңЁеҶ…еӯҳеҸ—йҷҗзҺҜеўғдёӯз®ҖеҚ•зҡ„иҪ»йҮҸзә§shell, дҪҶдёҺbash shellе®Ңе…Ёе…је®№ | | korn | дёҖз§ҚдёҺBourne shel1е…је®№зҡ„зј–зЁӢsell,дҪҶж”ҜжҢҒеҰӮе…іиҒ”ж•°з»„е’Ңжө®зӮ№иҝҗз®—зӯүдёҖдәӣй«ҳзә§зҡ„зј–зЁӢзү№жҖ§ | | tcsh | дёҖз§Қе°ҶcиҜӯиЁҖдёӯзҡ„дёҖдәӣе…ғзҙ еј•е…ҘеҲ°shellи„ҡжң¬дёӯзҡ„shell | | zsh | з§Қз»“еҗҲдәҶbash. tcshе’Ңkornзҡ„зү№жҖ§, еҗҢж—¶жҸҗдҫӣй«ҳзә§зј–зЁӢзү№жҖ§гҖҒе…ұдә«еҺҶеҸІж–Үд»¶е’Ңдё»йўҳеҢ–жҸҗзӨәз¬Ұзҡ„й«ҳзә§shell |

  3. LinuxжЎҢйқўзҺҜеўғпјҲеӣҫеҪўеҢ–з•ҢйқўGUIпјүпјҡзӣёеҜ№дәҺж–Үжң¬з•ҢйқўCLI

    • иҝҷдёҖйғЁеҲҶжҳҜеҸҜйҖүзҡ„пјҢе№¶дёҚжҳҜжүҖжңүзҡ„зЁӢеәҸйғҪйңҖиҰҒеӣҫеҪўеҢ–з•ҢйқўпјҢLinuxдёӯзӣёеҪ“дёҖеӨ§йғЁеҲҶзЁӢеәҸеҸҜд»ҘйҖҡиҝҮе‘Ҫд»ӨиЎҢз•ҢйқўдёҺз”ЁжҲ·дәӨдә’пјҢдҫӢеҰӮ[GROMACS]
    • KDEжЎҢйқўпјҡпјҲK Desktop EnvironmentпјҢKжЎҢйқўзҺҜеўғпјүзұ»дјјдәҺMicrosoft Windowsзҡ„еӣҫеҪўеҢ–жЎҢйқўзҺҜеўғпјҢдҫӢеҰӮ openSuSE LinuxеҸ‘иЎҢзүҲ
    • GNOMEжЎҢйқўпјҡпјҲthe GNU Network Object Model EnvironmentпјҢGNUзҪ‘з»ңеҜ№иұЎжЁЎеһӢзҺҜеўғпјүпјҢдҫӢеҰӮ CentOS 7 е’ҢUbuntu20 LinuxеҸ‘иЎҢзүҲ

  4. еә”з”ЁиҪҜд»¶пјҡйҖҡиҝҮGNUе·Ҙе…·е’ҢLinuxжЎҢйқўзҺҜеўғиҝҗиЎҢеңЁLinuxеҶ…ж ёд№ӢдёҠпјҢжҳҜLinuxзі»з»ҹзӣҙжҺҘдёҺз”ЁжҲ·дәӨдә’зҡ„дёҖйғЁеҲҶ

Linuxзҡ„иҪҜд»¶з®ЎзҗҶ вҖӢ

иҪҜд»¶еҢ…е’ҢеҢ…з®ЎзҗҶеҷЁ вҖӢ

  • LinuxдёӯеҫҲеӨҡиҪҜд»¶йғҪжҳҜд»Ҙжәҗд»Јз Ғж–№ејҸиҝӣиЎҢеҸ‘еёғзҡ„пјҢдҪҶжҳҜжәҗд»Јз ҒеҜ№жҷ®йҖҡз”ЁжҲ·жқҘи®Ізј–иҜ‘жҲҗдёҖдёӘеҸҜжү§иЎҢзҡ„ж–Үд»¶иҝҮзЁӢжҳҜйқһеёёз№Ғзҗҗзҡ„гҖӮжүҖд»ҘжңүдәӣиҪҜд»¶е•Ҷеё®жҲ‘д»¬жҠҠдә’иҒ”дёҠзҡ„ејҖжәҗиҪҜд»¶зј–иҜ‘жҲҗдәҶдәҢиҝӣеҲ¶е№¶дё”жҠҠе®ғжү“еҢ…жҲҗдёҖдёӘж–Үд»¶з„¶еҗҺжҲ‘д»¬еҸҜд»ҘзӣҙжҺҘз”ЁеҢ…з®ЎзҗҶеҷЁе°ҶиҪҜд»¶е•Ҷжү“еҢ…еҘҪзҡ„ж–Үд»¶зӣҙжҺҘиҝӣиЎҢе®үиЈ…гҖӮ
  • е®үиЈ…иҪҜд»¶зҡ„дёӨз§Қж–№ејҸ
    • д»Һжәҗз ҒпјҲзј–зЁӢд»Јз ҒпјүиҮӘиЎҢзј–иҜ‘
    • йҖҡиҝҮиҪҜд»¶еҢ…з®ЎзҗҶзі»з»ҹпјҢе·Із»Ҹзј–иҜ‘еҘҪзҡ„дәҢиҝӣеҲ¶е®үиЈ…зЁӢеәҸ
      • иҪҜд»¶еҢ…з®ЎзҗҶзі»з»ҹжҸҗдҫӣе®үиЈ…гҖҒжӣҙж–°гҖҒеҲ йҷӨеҸҠеҚёиҪҪиҪҜд»¶зҡ„ж–№жі•пјҢеҗҢж—¶жҸҗдҫӣеҜ№зі»з»ҹдёӯжүҖжңүиҪҜд»¶зҠ¶жҖҒдҝЎжҒҜзҡ„жҹҘиҜўгҖӮдёҚеҗҢзҡ„LinuxеҸ‘иЎҢзүҲдјҡжңүдёҚеҗҢзҡ„еҢ…з®ЎзҗҶеҷЁпјҢжҜ”иҫғеёёи§Ғзҡ„еҢ…з®ЎзҗҶе·Ҙе…·жңүDPKGгҖҒAPT(APT-GET)гҖҒRPMгҖҒYUMгҖҒDNFгҖҒPACMANзӯү

    • runеҢ…пјҡиҝҷз§Қж јејҸзҡ„еҢ…дёҺbinеҢ…зұ»дјјпјҢе®үиЈ…ж—¶д№ҹйңҖиҰҒз»ҷеҢ…еҠ дёҠеҸҜжү§иЎҢжқғйҷҗ
  2. иҪҜд»¶еҢ…зҡ„иҺ·еҸ–гҖҒиҪҜд»¶жәҗд»“еә“пјҡиҪҜд»¶еҢ…йңҖиҰҒдәӢе…Ҳе°Ҷжәҗд»Јз ҒиҝӣиЎҢзј–иҜ‘еҗҺжү“еҢ…еҪўжҲҗпјҢйҷӨдәҶеңЁеҸ‘иЎҢзүҲе®ҳж–№зҡ„д»“еә“дёӯжүҫпјҢиҺ·еҸ–еҢ…зҡ„йҖ”еҫ„еҰӮдёӢпјҡеҚҺдёәгҖҒйҳҝйҮҢдә‘гҖҒзҪ‘жҳ“гҖҒеӨ§еӯҰзӯүејҖжәҗй•ңеғҸ

е®үиЈ…дҪҚзҪ® вҖӢ

  • дё»иҰҒзҡ„е®үиЈ…дҪҚзҪ®жңүдёӨдёӘпјҢеҲҶеҲ«жҳҜ /opt зӣ®еҪ•е’Ң /usr/local зӣ®еҪ•гҖӮ
  • еңЁubuntuзі»з»ҹдёӯпјҢдёӢиҪҪзҡ„е®үиЈ…еҢ…иў«еӯҳж”ҫеңЁ/var/cache/apt/archivesзӣ®еҪ•дёӢгҖӮеҪ“жҲ‘д»¬дҪҝз”Ёapt-getжҲ–aptе‘Ҫд»ӨжқҘе®үиЈ…иҪҜд»¶ж—¶пјҢзі»з»ҹдјҡиҮӘеҠЁд»ҺиҝҷдёӘзӣ®еҪ•дёӯжҹҘжүҫзӣёеә”зҡ„еҢ…пјҢе№¶иҝӣиЎҢе®үиЈ…гҖӮиҝҷдёӘзӣ®еҪ•жҳҜзі»з»ҹй»ҳи®Өзҡ„иҪҜд»¶еҢ…зј“еӯҳзӣ®еҪ•пјҢе®ғдҝқеӯҳдәҶжҲ‘д»¬е·Із»ҸдёӢиҪҪиҝҮзҡ„еҢ…пјҢд»Ҙдҫҝе°ҶжқҘеҶҚж¬Ўе®үиЈ…ж—¶иғҪеӨҹеҝ«йҖҹжүҫеҲ°гҖӮ/var/cache/apt/archivesзӣ®еҪ•дёӢзҡ„е®үиЈ…еҢ…йғҪжҳҜд»Ҙ.debдёәеҗҺзјҖзҡ„ж–Үд»¶пјҢиҝҷжҳҜubuntuзі»з»ҹзҡ„иҪҜд»¶еҢ…ж јејҸгҖӮ
  • йҖҡиҝҮеңЁз»Ҳз«ҜдёӯиҝҗиЎҢе‘Ҫд»Өsudo apt-get install иҪҜд»¶еҢ…еҗҚз§°пјҢеҸҜд»Ҙд»Һ Ubuntu иҪҜд»¶д»“еә“дёӯзӣҙжҺҘдёӢиҪҪе’Ңе®үиЈ…иҪҜд»¶гҖӮиҝҷдәӣиҪҜд»¶еҢ…дјҡиў«е®үиЈ…еңЁзі»з»ҹзҡ„вҖң/usrвҖқзӣ®еҪ•дёӢзҡ„зү№е®ҡдҪҚзҪ®пјҢе…·дҪ“дҪҚзҪ®дјҡдҫқжҚ®иҪҜд»¶зҡ„зұ»еһӢе’Ңз”ЁйҖ”иҖҢжңүжүҖдёҚеҗҢгҖӮ
  • дёҖдәӣиҪҜд»¶еҸҜд»ҘйҖҡиҝҮз¬¬дёүж–№иҪҜд»¶жәҗиҝӣиЎҢе®үиЈ…пјҢжҜ”еҰӮ PPAпјҲPersonal Package ArchivesпјүжҲ–иҖ…йҖҡиҝҮиҪҜд»¶ејҖеҸ‘иҖ…зҡ„е®ҳж–№зҪ‘з«ҷжҸҗдҫӣзҡ„дёӢиҪҪй“ҫжҺҘпјҢе№¶дҪҝз”ЁйҖӮеҪ“зҡ„е‘Ҫд»ӨиҝӣиЎҢе®үиЈ…пјҢиҪҜд»¶еҢ…дјҡиў«еӯҳеӮЁеңЁзі»з»ҹзҡ„вҖң/optвҖқзӣ®еҪ•дёӢпјҢйҖҡеёёдјҡеҲӣе»әдёҖдёӘдёҺиҪҜд»¶еҗҚз§°зӣёеҜ№еә”зҡ„ж–Үд»¶еӨ№гҖӮ
  • з”ЁжҲ·д№ҹеҸҜд»ҘйҖҡиҝҮзӣҙжҺҘдёӢиҪҪиҪҜд»¶еҢ…е№¶жүӢеҠЁе®үиЈ…зҡ„ж–№ејҸжқҘиҺ·еҫ—иҪҜд»¶гҖӮиҝҷдәӣиҪҜд»¶еҢ…еҸҜиғҪжҳҜйҖҡз”Ёзҡ„дәҢиҝӣеҲ¶ж–Үд»¶пјҢд№ҹеҸҜиғҪжҳҜз»ҸиҝҮзј–иҜ‘зҡ„жәҗд»Јз ҒгҖӮз”ЁжҲ·еҸҜд»ҘйҖүжӢ©е°ҶиҪҜд»¶еҢ…еӯҳеӮЁеңЁд»»ж„ҸдҪҚзҪ®пјҢз„¶еҗҺдҪҝз”Ёзӣёеә”зҡ„е‘Ҫд»ӨиҝӣиЎҢе®үиЈ…гҖӮ
  • йҖҡиҝҮеҢ…з®ЎзҗҶеҷЁе®үиЈ…зҡ„иҪҜд»¶йҖҡеёёдјҡеңЁзі»з»ҹзҡ„вҖң/usr/binвҖқзӣ®еҪ•дёӢеҲӣе»әдёҖдёӘеҸҜжү§иЎҢж–Үд»¶гҖӮз”ЁжҲ·еҸҜд»ҘеңЁз»Ҳз«ҜдёӯзӣҙжҺҘи°ғз”ЁиҜҘж–Үд»¶жқҘеҗҜеҠЁиҪҜд»¶гҖӮжӯӨеӨ–пјҢдёҖдәӣиҪҜд»¶иҝҳдјҡеңЁвҖң/usr/share/applicationsвҖқзӣ®еҪ•дёӢеҲӣе»әдёҖдёӘ .desktop ж–Үд»¶пјҢиҝҷж ·з”ЁжҲ·е°ұеҸҜд»ҘеңЁеӣҫеҪўз•Ңйқўдёӯжӣҙж–№дҫҝең°еҗҜеҠЁиҪҜд»¶дәҶгҖӮ(ubuntuзі»з»ҹеңЁ~/.local/share/applications/)
  • еҰӮжһңжҳҜиҮӘиЎҢд»Һжәҗз Ғзј–иҜ‘е®үиЈ…зҡ„зЁӢеәҸпјҢйҖҡеёёдёҚдјҡе°ҶеҸҜжү§иЎҢж–Үд»¶ж”ҫеңЁвҖң/usr/binвҖқзӣ®еҪ•пјҢиҰҒеңЁз»Ҳз«ҜдёӯзӣҙжҺҘи°ғз”ЁйңҖиҰҒж·»еҠ зҺҜеўғеҸҳйҮҸ

зҺҜеўғеҸҳйҮҸ вҖӢ

  • зҺҜеўғеҸҳйҮҸз”ЁдәҺжҢҮе®ҡж“ҚдҪңзі»з»ҹиҝҗиЎҢзҺҜеўғзҡ„дёҖдәӣеҸӮж•°пјҢеҰӮPATHгҖҒHOMEгҖҒUSERгҖҒHISTSIZEгҖҒLOGNAMEгҖҒHOSTNAMEгҖҒSHELLгҖҒLANGUAGEгҖҒMAILе’ҢPS1зӯүгҖӮиҝҷдәӣеҸҳйҮҸеҸҜд»ҘеңЁзі»з»ҹзә§еҲ«жҲ–з”ЁжҲ·зә§еҲ«и®ҫзҪ®пјҢе№¶дё”еҸҜд»ҘеҲҶдёәж°ёд№…жҖ§е’Ңдёҙж—¶жҖ§дёӨз§ҚгҖӮзі»з»ҹзҺҜеўғеҸҳйҮҸеҜ№зі»з»ҹдёӯжүҖжңүз”ЁжҲ·йғҪжңүж•ҲпјҢиҖҢз”ЁжҲ·зҺҜеўғеҸҳйҮҸеҸӘеҜ№зү№е®ҡз”ЁжҲ·жңүж•ҲгҖӮ
  • з”ЁжҲ·еҸҜд»ҘйҖҡиҝҮзј–иҫ‘дёӘдәәй…ҚзҪ®ж–Үд»¶пјҲеҰӮ~/.bashrcжҲ–~/.profileпјүжқҘи®ҫзҪ®з”ЁжҲ·зҡ„зҺҜеўғеҸҳйҮҸпјҢиҝҷдәӣж–Үд»¶йҖҡеёёдҪҚдәҺз”ЁжҲ·зҡ„дё»зӣ®еҪ•дёӢгҖӮ
  • зі»з»ҹзҺҜеўғеҸҳйҮҸйҖҡеёёи®ҫзҪ®еңЁ/etc/profileж–Үд»¶дёӯпјҢиҝҷдёӘж–Үд»¶еңЁз”ЁжҲ·зҷ»еҪ•ж—¶дјҡиҮӘеҠЁиҜ»еҸ–пјҢд»Ҙи®ҫзҪ®зі»з»ҹзә§зҡ„зҺҜеўғеҸҳйҮҸгҖӮ
  • жӢ“еұ•йҳ…иҜ»пјҡlinuxзҺҜеўғеҸҳйҮҸ

иҝҗиЎҢиҪҜд»¶ж–№ејҸ вҖӢ

  • йҷӨдәҶзі»з»ҹиҮӘеёҰзҡ„еҸҜжү§иЎҢзЁӢеәҸпјҢзӣҙжҺҘиҫ“е…ҘиҮӘе·ұзҡ„зЁӢеәҸгҖҒж–°е®үиЈ…зҡ„зЁӢеәҸпјҢзі»з»ҹжІЎеҠһжі•иҺ·еҸ–еҲ°иҜҘжү§иЎҢзЁӢеәҸзҡ„жүҖеңЁж–Үд»¶и·Ҝеҫ„пјҢеҚізі»з»ҹдёҚзҹҘйҒ“иҝҷдёӘзЁӢеәҸеңЁе“ӘйҮҢпјҢж— жі•иҝҗиЎҢгҖӮ
  • жү“еҚ°дёҖдёӢеҪ“еүҚзі»з»ҹ PATH зҺҜеўғеҸҳйҮҸпјҡecho $PATH
  • е°ҶиҰҒжү§иЎҢзҡ„ж–Үд»¶жүҖеңЁзҡ„и·Ҝеҫ„ж·»еҠ еҲ°иҜҘеҸҳйҮҸдёӯпјҡPATH=$PATH:еҸҜжү§иЎҢзЁӢеәҸзҡ„ж–Үд»¶жүҖеңЁи·Ҝеҫ„
