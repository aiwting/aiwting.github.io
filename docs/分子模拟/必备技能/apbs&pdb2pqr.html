APBS&PDB2PQR ​

apbs和pdb2pqr计算生物大分子静电的一般使用流程: 起始pdb结构 ->pdb2pqr ->apbs ->可视化

结构准备 ​

  • 首先下载或构建分子结构文件, 如pdb文件.
  • 想要对生物大分子进行静电计算, 还需要提供每个原子所带电荷及原子半径等信息.
    • 电荷用来为求解泊松-波尔兹曼(PB)方程提供生物分子的电荷分布
    • 半径则用来构建电介质和离子的可及度函数
  • 存储电荷和半径信息的两种文件格式
    • PQR格式: 将PDB格式结构文件中的occupancy和temperature列替换为电荷(Q)和半径(R)信息
    • XML格式: 可扩展性更强

由pdb生成pqr ​

  • 在conda环境中使用pdb2pqr
bash
pdb2pqr 1lse.pdb 1lse.pqr --drop-water --ff AMBER --ffout AMBER --apbs-input 1lse_apbs.in --pdb-output 1lse_4.6.pdb --titration-state-method propka --with-ph 4.6
  • -ff 使用指定力场中的参数, 可以是{AMBER,CHARMM,PARSE,TYL06,PEOEPB,SWANSON}中的一种
  • -ffout 使用指定力场中的残基名称作为输出(一些残基不同的电离状态在不同力场中的名称不同)
  • --apbs-input 输出用于 apbs 计算的输入文件.
  • --titration-state-method propka和 --with-ph 表示使用 propka 工具计算指定 pH 下时的可电离残基的pKa值
  • -pdb-output 输出更改质子化状态后的pdb结构文件.

使用自定义力场 ​

  • 如果你的结构中含有非标残基, 需要提供这些特殊残基的每个原子的相关参数才能使用pdb2pqr
  • 编辑 PDB2PQR 自带的力场数据文件（dat/AMBER.DAT）并添加参数.
  • 或者重新生成一个全新的力场数据文件.
  • Extending PDB2PQR — documentation

通过tpr文件生成pqr ​

  • GROMACS的tpr文件中包含了原子电荷和半径信息.
  • 使用editconf命令可以生成qpr文件, 但这种做法致命缺点是原子电荷保留精度太低, 只有两位小数
bash
gmx editconf -f xxx.tpr -mead xxx.pqr

根据pqr计算静电势 ​

  • VMD中集成了apbs插件
  • pymol中也集成了apbs插件
  • 但版本古老, 我觉得很不好用, 不比命令行操作清晰明了.
  • 在conda环境中使用apbs
bash
apbs --output-file=1lse_apbs.data 1lse_apbs.in > 1lse_apbs.log
  • 输出的.dx文件，包含了静电势格点数据，可加载到VMD中可视化
