PACKMOL简介 ​

  • Packmol是一个构建分子动力学模拟初始结构的工具, 构建分子动力学模拟初始模型的软件。
  • 由Sánchez和Chomilier开发，依据设定的条件，在空间范围内填充分子，形成不同的体系，如球状、管状、层状等体系。能够保证短程相互作用斥力不会影响模拟。
  • #免费 #开源 #windows #linux
  • 官网: PACKMOL
  • packmol基础教程 (qq.com)
  • Packmol的使用
  • Packmol程序通过设置inp文件控制结构生成
  • 使用命令: packmol < xxx.inp

.inp输入文件内容： ​

.inp
tolerance 2.0           不同分子的原子间距不小于2Å
output test.pdb
filetype pdb
structure xxx1.pdb       分子结构
    number 111
    inside cube 0. 0. 0. 40.        空间规则：随机分布在最小坐标(0,0,0)最大最大坐标(40,40,40)的立方体内
end structure
structure xxx2.pdb
    inside cube 0. 0. 0. 20.
    atoms 9 10                      限制原子9，10的位置
        inside box 0. 0. 15. 20. 20. 20.
    end atoms
end structure

空间约束规则 ​

  • 有几种类型的约束既可以应用于整个分子，也可以应用于分子的一部分。这些约束定义了分子必须位于溶液中的空间区域。可以以这种方式构建非常有序的系统。约束条件包括： 1.固定分子fixed x y z a b g 2.放置在立方体内inside cube xmin ymin zmin d 3.放置在立方体外outside cube xmin ymin zmin d 4.放置在盒子内inside box xmin ymin zmin xmax ymax zmax 5.放置在盒子外outside box xmin ymin zmin xmax ymax zmax 6.放置在球内/外: (x−a)^2+(y−b)^2+(z−c)^2−d^2=0 inside sphere a b c d
    outside sphere a b c d 7.椭球 inside/outside ellipsoid a1 b1 c1 a2 b2 c2 d 8.above (or below) plane: ax+by+cz−d=0 above plane 2.5 3.2 1.2 6.2
    below plane 2.5 3.2 1.2 6.2 9.inside (or outside) cylinder圆柱体: p=(a1,b1,c1)+t×(a2,b2,c2) inside cylinder a1 b1 c1 a2 b2 c2 d loutside cylinder a1 b1 c1 a2 b2 c2 d l 10.Constrain rotations: 限制每种类型的所有分子的旋转，使它们在空间中具有一些平均取向。 constrain_rotation x 180. 20.
    constrain_rotation y 180. 20.
    constrain_rotation z 180. 20. 11.over (or below) xygauss (gaussian surface) over xygauss a1 b1 a2 b2 c hbelow xygauss a1 b1 a2 b2 c h 12.设置不同分子间原子半径
.inp
structure water.pdb
  number 500
  inside box 0. 0. 0. 30. 30. 30.
  radius 1.5           #原子的半径为 1.5Å，意味着分子内的距离为 3.0Å
end structure

周期性边界条件 ​

如果您的系统在 100Å 盒子中模拟，则将分子约束在98Å盒子内，当构建带有 PBC 的实际模拟盒时，图像将彼此分开2Å。边界处将有一个空白空间，但随着能量最小化和平衡，这将很容易消失。

溶剂化大分子 ​

  • solvate.tcl PROTEIN.pdb该脚本将创建一个名为 packmol_input.inp 的输入文件，包括框的大小和建议要使用的周期性边界条件尺寸
  • packmol < packmol_input.inp大分子将被 15Å 的水壳溶剂化，离子保持系统中性，生理NaCl浓度0.16M。
  • solvate.tcl structure.pdb -shell 15. -charge +5 -density 1.0 -i pack.inp -o solvated.pdb残基电性：Arg and Lys as +1 and Glu and Asp as -1. -noions 不添加任何离子，只添加水
